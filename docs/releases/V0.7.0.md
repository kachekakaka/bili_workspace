# bili_workspace v0.7.0

V0.7.0 完成前端内部结构整理，不改变下载、API、权限、标签或数据库模型。

## 主要变化

- 全部页面迁移到原生 `.mjs` ES Modules 和统一生命周期；
- Router 在切页时确定性取消旧请求、订阅和事件；
- API、Session、Context、Modal、Toast、Confirm 和 TaskStream 收敛为单一实现；
- Search 保留当前页请求、本地筛选和最多下一页预载；
- Library 保留完整筛选、批量操作、改分组、Range 播放和续播；
- Tasks 与 Dashboard 共享唯一 SSE；
- 删除旧 App Shell、全部 enhancement/overlay、Legacy Bridge 和版本专用 CSS；
- 最终入口为一个应用模块和四层语义 CSS；
- Windows/Linux 自检覆盖 `.js/.mjs` 与 Node tests；
- `.mjs` 纳入运行构建指纹。

## 兼容性

- SQLite schema 仍为 v4；
- API 和权限矩阵不变；
- Windows 与 Docker 持久化目录不变；
- 可从 v0.6.2 原地升级；
- 回滚到 `v0.6.2` 时无需数据库降级，但应先停止服务并备份现场。

## 验证

发布前已要求通过 Python 3.11/3.12/3.13、compileall、Ruff、完整 pytest、Node syntax/tests、五档 Playwright、Windows `verify.bat`、Docker build 和镜像内版本冒烟。

完整验收见 [`docs/V0.7功能与验收.md`](../V0.7功能与验收.md)。
