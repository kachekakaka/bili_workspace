# UI Issues

## V0.6.2 需要处理的问题

## P0

### 控件尺寸不统一

现象：

- 下拉框高度明显大于按钮。
- 输入框和操作按钮视觉比例不一致。

处理：

统一桌面端普通控件高度。

### Prompt 交互

现象：

- 用户修改显示名使用浏览器 prompt。

处理：

统一 Modal。

### 账号页面职责混合

现象：

- Bilibili 登录和网站用户密码管理显示在一起。

处理：

拆分账号管理和 Bilibili 账号。

## P1

### 下拉选择

现象：

- 动态数据使用普通下拉不易查找。

处理：

增加搜索选择能力。

### 设置页面

现象：

- 高级配置和普通配置混合。

处理：

基础设置默认显示，高级设置折叠。

## P2

### 页面细节

- 减少无意义留白。
- 优化卡片高度。
- 压缩筛选区域。

## 非目标

- 标签系统重构。
- 下载流程重写。
- 大规模前端架构调整。
