## 当前正式入口

- 唯一应用脚本：`web/assets/app/main.mjs`；
- 唯一允许的第三方全局：`window.QRCode`；
- CSS：
  - `web/assets/styles/tokens.css`
  - `web/assets/styles/base.css`
  - `web/assets/styles/components.css`
  - `web/assets/styles/pages.css`
- 页面：全部位于 `web/assets/app/pages/*.mjs`；
- 版本恢复提示：`web/assets/app/core/version-check.mjs`；
- 任务实时流：`web/assets/app/core/task-stream.mjs`。

## 最终固定门禁

- `index.html` 只有一个应用 JS 入口；
- 不存在 `enhancements-*.js`、`enhancements-*.css`、`ui-v*.css`；
- 不存在 `web/assets/app/legacy/` 内容；
- 不存在 `window.BiliLegacyApp` 或 `window.BiliEnhancements`；
- 页面无 `MutationObserver`、捕获阶段抢占、原生 `prompt()` 或 `confirm()`；
- API、Session、Modal、Toast 和 TaskStream 各只有一份；
- 每个路由只有一个 renderer；
- 最终唯一允许保留的第三方全局是 `window.QRCode`。
