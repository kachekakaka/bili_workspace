## 固定门禁

- PR 1 不得在生产入口加载 `web/assets/app/` 下的新模块；
- 不得新增 `enhancements-v070-*` 或 `ui-v070-*` 文件；
- 新纯模块不得访问 `window`、`document`、`MutationObserver` 或 `window.BiliEnhancements`；
- 每个旧 renderer/overlay 只能在同一 PR 已有正式替代实现和行为测试时删除；
- `window.BiliLegacyApp` 与 `window.BiliEnhancements` 必须在 PR 7 删除；
- 最终唯一允许保留的第三方全局是 `window.QRCode`。
