## 使用规则

- 新窗口必须先读取最新 `main`、当前计划、Review、open PR、feature 分支和 CI；
- 一次只实施一个 PR 阶段；
- 每阶段必须使用分支、PR、CI green、merge main；
- 未合并内容不得写进当前主线功能清单；
- 计划发生变化时必须先更新方案并重新 Review。
