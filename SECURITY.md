# Security Policy

## Supported version

当前维护版本为 V0.5.x。

## Reporting a vulnerability

请不要在公开 Issue 中粘贴真实 Cookie、`BBDown.data`、初始化令牌、管理员会话、域名后台信息或 NAS 路径截图。

发现认证绕过、任意文件读写、路径穿越、命令执行、Cookie 泄露或媒体越权访问问题时，应通过仓库的私密安全报告渠道提交。报告请包含：

- 受影响版本；
- 最小复现步骤；
- 预期与实际行为；
- 已脱敏的日志或请求；
- 对数据和部署的影响。

在修复发布前，请避免公开可直接利用的细节。
